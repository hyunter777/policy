VANTA PLAYER 프로젝트

공개 페이지

이 페이지는 VANTA PLAYER의 공개 개인정보처리방침 페이지입니다.

최신 버전은 이 주소에서 확인할 수 있습니다.

1. VANTA PLAYER가 하는 일

VANTA PLAYER는 Android 기기에서 로컬 파일과 네트워크 라이브러리를 재생할 수 있는 미디어 플레이어입니다.

  • 기기 안의 로컬 비디오 및 오디오를 탐색하고 재생할 수 있습니다.
  • SMB, WebDAV, FTP, SFTP 같은 사용자가 직접 등록한 원격 라이브러리를 탐색할 수 있습니다.
  • 최근 재생 이력, 이어보기 위치, 재생 환경 설정을 기기 안에 저장할 수 있습니다.
  • 고급 재생 흐름을 위해 앱 안에서 내부 저장소 탐색기를 제공합니다.

2. 앱이 접근할 수 있는 데이터

로컬 미디어 및 파일 접근

사용자가 기기 안의 콘텐츠를 탐색하고 재생할 수 있도록 로컬 미디어 파일과 관련 메타데이터에 접근할 수 있습니다.

현재 빌드가 넓은 저장소 접근 권한을 사용할 경우, 앱 내 탐색기와 재생 흐름을 위해 아래 정보에도 접근할 수 있습니다.

  • 폴더 이름
  • 파일 이름
  • 숨김 폴더
  • 자막 sidecar 파일
  • 재생과 탐색에 필요한 기타 로컬 파일 경로

원격 라이브러리 연결 정보

사용자가 원격 라이브러리를 추가하면 아래와 같은 연결 정보를 로컬에 저장하고 사용할 수 있습니다.

  • 서버 주소
  • 사용자 이름
  • 비밀번호
  • 호환 프로토콜에서 필요한 domain 값

이 정보는 사용자가 직접 선택한 원격 소스에 연결하기 위한 용도로만 사용됩니다.

재생 및 환경설정 데이터

앱은 다음 정보를 기기 안에 저장할 수 있습니다.

  • 최근 재생 이력
  • 이어보기 위치
  • 자막 설정
  • 재생 설정
  • 저장된 원격 라이브러리 목록

3. 데이터 사용 목적

VANTA PLAYER는 접근한 데이터를 아래 목적에 사용합니다.

  • 로컬 미디어 탐색 및 재생
  • 사용자가 등록한 원격 소스 연결 및 탐색
  • 최근 재생과 환경설정 유지
  • 자막 및 탐색기 기능 지원
  • 호환성 및 안정성 문제 진단
  • 광고가 활성화된 빌드의 경우 광고 및 동의 흐름 지원

4. 광고 및 동의

광고가 활성화된 빌드에서 VANTA PLAYER는 Google Mobile Ads SDK와 Google User Messaging Platform(UMP)을 사용할 수 있습니다.

  • 필요한 지역에서 사용자 동의 요청 또는 갱신
  • 허용된 앱 화면에서 광고 표시
  • 필요한 경우 privacy options 진입점 제공

현재 제품 기준으로 광고는 재생 화면이 아닌 Home/non-playback 영역을 대상으로 설계되어 있으며, fullscreen playback을 광고가 방해하지 않도록 유지하는 것이 기준입니다.

Google의 광고 및 동의 SDK는 IP 주소, 앱 상호작용, 진단 정보, 광고/기기 식별자 같은 데이터를 처리할 수 있습니다. 이 부분은 Google의 정책과 공개 자료를 따릅니다.

5. 데이터 공유

VANTA PLAYER는 사용자의 로컬 미디어 콘텐츠 자체를 판매하지 않습니다.

광고가 활성화된 빌드에서는 Google Mobile Ads SDK와 consent flow의 일부로 Google 및 관련 광고 시스템이 일부 데이터를 처리할 수 있습니다.

사용자가 원격 소스를 추가한 경우, 네트워크 요청은 사용자가 직접 지정한 엔드포인트로만 전송됩니다.

6. 보관 및 삭제

기기 안에 저장된 데이터는 다음 경우까지 유지될 수 있습니다.

  • 사용자가 앱 데이터를 삭제하는 경우
  • 사용자가 앱 안에서 항목이나 설정을 제거하는 경우
  • 앱을 삭제하는 경우

원격 라이브러리 정보, 재생 이력, 설정은 기기 내 사용성과 연속성을 위해 저장됩니다. 제3자 SDK 데이터 보관은 해당 SDK 제공자의 정책을 따를 수 있습니다.

7. 보안

VANTA PLAYER는 데이터 사용을 재생, 탐색, 광고/동의 기능 범위로 제한하기 위해 합리적인 수준의 노력을 기울입니다. 다만 저장 및 전송 방식이 완전히 안전하다고 보장할 수는 없습니다.

8. 정책 변경

앱 동작, 포함된 SDK, 법적 요구사항이 바뀌면 이 정책이 업데이트될 수 있습니다. 최신 버전은 이 페이지에 게시됩니다.

English Summary

VANTA PLAYER is an Android media player for local playback and user-configured network libraries. The app may access local media, local file paths needed for in-app browsing, remote connection details supplied by the user, playback history, resume position, subtitle preferences, and saved settings.

If ads are enabled in the build, VANTA PLAYER may use Google Mobile Ads SDK and Google User Messaging Platform (UMP) for consent and advertising on approved non-playback surfaces. Fullscreen playback is not intended to be interrupted by ads in the current product baseline.
